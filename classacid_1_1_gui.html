class UiObject
A representation of a object this is rendered to a screen. This object is contained in a parent and has children. The screen object has a few values that allow for it to be positioned and scaled, along with other variables that are used when rendering. This class can be extended to create a representation for GUI images, fonts, etc.

void acid::Gui::SetNinePatches(const Vector4f& ninePatches)

Parameters
ninePatches The values, x/y being to top left corner and z/w bottom right for the scalable section.

Sets nine the patches value size, if zero it will be disabled. This value represents the top-left and bottom-right content bounds. 9-patch/9-slicing allows for a single section of a image to be scale with corners and edges kept in the screens aspect ratio.
