Enum documentation

enum class acid::HttpRequest::Method

Enumerate the available HTTP methods for a request, https://developer.mozilla.org/en-US/docs/Web/HTTP/Methods.

Enumerators
Get

Request in get mode, standard method to retrieve a page.
Post

Request in post mode, usually to send data to a page.
Head

Request a page's header only.
Put

Request in put mode, useful for a REST API.
Delete

Request in delete mode, useful for a REST API.
Options

Request in options mode, describes the communication options for the page.
Patch

Request applies partial modifications to a page.
Trace

Request performs a message loop-back test along the path to the target resource.
Connect

Request establishes a tunnel to the server page.

void acid::HttpRequest::SetField(const std::string& field, const std::string& value)

Parameters
field Name of the field to set.
value Value of the field.

Set the value of a field. The field is created if it doesn't exist. The name of the field is case-insensitive. By default, a request doesn't contain any field (but the mandatory fields are added later by the HTTP client when sending the request).

void acid::HttpRequest::SetMethod(Method method)

Parameters
method Method to use for the request.

Set the request method. See the Method enumeration for a complete list of all the available methods. The method is HTTP_REQUEST_METHOD_GET by default.

void acid::HttpRequest::SetUri(const std::string& uri)

Parameters
uri URI to request, relative to the host.

Set the requested URI. The URI is the resource (usually a web page or a file) that you want to get or post. The URI is "/" (the root page) by default.

void acid::HttpRequest::SetHttpVersion(uint32_t major, uint32_t minor)

Parameters
major Major HTTP version number.
minor Minor HTTP version number.

Set the HTTP version for the request. The HTTP version is 1.0 by default.

void acid::HttpRequest::SetBody(const std::string& body)

Parameters
body Content of the body.

Set the body of the request. The body of a request is optional and only makes sense for POST requests. It is ignored for all other methods. The body is empty by default.
