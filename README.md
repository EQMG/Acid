# Flounder v0.1
Flounder is a cross platform high speed C++ OpenGL 2D/3D game engine.

Libraries:
 * GLFW / GLFW for OpenGL
 * STB image loading
 * OpenAL with alut sound loaders

Supported platforms:
 * Windows
 * MacOS
 * Linux

Rendering backends:
 * OpenGL
 
Audio backends:
 * OpenAL

Features:
 * Cross platform
 * High speed
 * 2D and 3D support
 * Deferred rendering
 * Event and tasks systems
 * Component entity system
 * 3D shadow mapping
 * 3D skyboxes
 * Avalable visual drivers and maths classes
 * Flexable inputs
