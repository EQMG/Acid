Status acid::TcpSocket::Connect(const IpAddress& remoteAddress, uint16_t remotePort, const Time& timeout = 0s)

Parameters
remoteAddress Address of the remote peer.
remotePort Port of the remote peer.
timeout Optional maximum time to wait.
Returns Status code.

Connect the socket to a remote peer. In blocking mode, this function may take a while, especially if the remote peer is not reachable. The last parameter allows you to stop trying to connect after a given timeout. If the socket is already connected, the connection is forcibly disconnected before attempting to connect again.

void acid::TcpSocket::Disconnect()

Disconnect the socket from its remote peer. This function gracefully closes the connection. If the socket is not connected, this function has no effect.

Status acid::TcpSocket::Send(const void* data, std::size_t size)

Parameters
data Pointer to the sequence of bytes to send.

Returns Status code.

Send raw data to the remote peer. To be able to handle partial sends over non-blocking sockets, use the send(const void*, std::size_t, std::size_t&) overload instead. This function will fail if the socket is not connected.

Status acid::TcpSocket::Receive(void* data, std::size_t size, std::size_t& received)

Parameters
data Pointer to the array to fill with the received bytes.
size Maximum number of bytes that can be received.

Receive raw data from the remote peer. In blocking mode, this function will wait until some bytes are actually received. This function will fail if the socket is not connected.

Status acid::TcpSocket::Send(Packet& packet)

Parameters
packet Packet to send.
Returns Status code.

Send a formatted packet of data to the remote peer. In non-blocking mode, if this function returns SOCKET_STATUS_PARTIAL, you must retry sending the same unmodified packet before sending anything else in order to guarantee the packet arrives at the remote peer uncorrupted. This function will fail if the socket is not connected.

Status acid::TcpSocket::Receive(Packet& packet)

Parameters
packet Packet to fill with the received data.
Returns Status code.

Receive a formatted packet of data from the remote peer. In blocking mode, this function will wait until the whole packet has been received. This function will fail if the socket is not connected.
